# Titanic Data Analysis Pipeline
### ML Foundations Capstone Project

This project is a complete data analysis pipeline built as part of the ML Foundations Bootcamp. It covers everything from raw data cleaning to feature engineering and exploratory data analysis.

## Project Overview
The goal of this project is to take the messy Titanic dataset and transform it into clean, useful insights by performing:
* **Phase 1:** Data Cleaning and Outlier Handling.
* **Phase 2:** Feature Engineering (Encoding, Scaling, and New Features).
* **Phase 3:** Exploratory Data Analysis (EDA) and Mathematical Analysis.

## Key Features
* **Clean Python Code:** Organized into sequential Jupyter Notebooks.
* **Feature Engineering:** One-hot encoding, feature scaling, and interaction features.
* **Data Visualization:** Distribution plots, correlation heatmaps, and boxplots.
* **Bonus Dashboard:** A combined 4-chart visualization for quick insights.
* **Math Basics:** Manual calculation of Mean, Standard Deviation, and Cosine Similarity using NumPy.

## Project Structure
* `01_cleaning.ipynb`: Loads data and handles missing values.
* `02_features.ipynb`: Creates and transforms features for analysis.
* `03_eda.ipynb`: Visualizes data and includes mathematical calculations.
* `report.md`: A summary report of findings and decisions.
* `requirements.txt`: List of necessary Python libraries.
* `data/`: Folder containing raw and cleaned datasets.

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run notebooks in order: `01` -> `02` -> `03`.
